# HAU Library Management System

Ứng dụng quản lý thư viện dành cho Trường Đại học Kiến Trúc Hà Nội (HAU), hỗ trợ quản lý sách, danh mục, người dùng và hoạt động mượn/trả sách.

## Tính năng chính

* Quản lý thông tin sách và danh mục sách.
* Tìm kiếm sách theo tên, tác giả hoặc thể loại.
* Quản lý người dùng và phân quyền.
* Quản lý mượn, trả và gia hạn sách.
* Thống kê và báo cáo hoạt động thư viện.
* Giao diện thân thiện trên Web và Android.

## Công nghệ sử dụng

### Frontend

* React
* TypeScript
* Vite

### Backend

* Node.js
* TypeScript

### Database

* MySQL

### Mobile

* Capacitor Android

## Cài đặt

```bash
npm install
npm run dev
```

## Cấu trúc dự án

```text
backend/          # Các service xử lý nghiệp vụ
services/         # Các API service phía client
database/         # Script cơ sở dữ liệu
android/          # Ứng dụng Android (Capacitor)
```
